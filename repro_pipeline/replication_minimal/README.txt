Replication Minimal Package
===========================

This folder contains a minimal, self-contained set of files to reproduce the core outputs:

- plot_scaled_overlay_minimal_smooth_all_to_2025_06.png
- plot_shipments_loess_policy_lines.png
- plot_seizures_loess_event_lines.png
- summary_statistics_through_2025_06.csv

Folder structure
----------------

- data/
  - monthly_input_with_rolling_overdose.csv
    - Monthly panel containing shipments (tx_raw), fentanyl seizures (seizure_lbs_raw), and rolling CDC overdose counts.
  - overdoseDeathsData_cleaned.csv
    - CDC synthetic opioid overdose series in rolling 12-month form.
  - policy_table_updated_all.csv
    - Policy implementation month-year table used for shipment-policy line overlays.
  - overdose_raw_from_rolling.csv
    - Generated by Script 01 (monthly overdose counts recovered from rolling counts).

- scripts/
  - 01_transform_overdose_rolling12_to_monthly.R
    - Reads overdoseDeathsData_cleaned.csv and reconstructs monthly overdose counts.
  - 02_make_main_plot.R
    - Creates the main paper figure: plot_scaled_overlay_minimal_smooth_all_to_2025_06.png.
  - 03_make_supplemental_outputs.R
    - Reads monthly_input_with_rolling_overdose.csv, overdose_raw_from_rolling.csv, and policy_table_updated_all.csv.
    - Creates the supplemental outputs:
      - plot_shipments_loess_policy_lines.png
      - plot_seizures_loess_event_lines.png
      - summary_statistics_through_2025_06.csv

- output/
  - figures/
  - tables/

How to run
----------

From the GitHub repository root (cnx-fentanyl-repro-pipeline), run:

1) cd cnx-fentanyl-repro-pipeline
2) Rscript repro_pipeline/replication_minimal/scripts/01_transform_overdose_rolling12_to_monthly.R
3) Rscript repro_pipeline/replication_minimal/scripts/02_make_main_plot.R
4) Rscript repro_pipeline/replication_minimal/scripts/03_make_supplemental_outputs.R

Outputs are written to:

- repro_pipeline/replication_minimal/output/figures/
- repro_pipeline/replication_minimal/output/tables/
